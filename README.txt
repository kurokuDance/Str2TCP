Тестовое задание.

Для запуска в папках prog1 и prog2 ноходяться ярлики для запуска программ.
Для компиляции программ в папках prog1 и prog2 ноходяться cmakelists.txt.

Рекомендуется  сначала запускать программу №2(сервер), а затем программу №1(клиент).

Для примера, число на выходе кратное 32: 111294365 (111KB3KB5KB9 = 128).
